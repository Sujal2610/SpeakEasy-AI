# SpeakEasy-AI
A  Voice-Controlled Virtual Assistant coded in Python
It is a lightweight voice-controlled assistant built using Python. It listens to your voice commands and performs tasks like web browsing, Wikipedia search, telling the time, sending emails, playing music, and launching applicationsâ€”making your desktop experience hands-free and interactive.

# Features
- Voice recognition using SpeechRecognition
- Text-to-speech feedback with pyttsx3
- Wikipedia search and summary reading
- Open popular websites like YouTube, Google, Stack Overflow
- Tell the current time
- Play music from your local directory
- Launch custom applications
-Send emails using Gmail SMTP

# Tech Stack
-Python 3.x
-pyttsx3
-SpeechRecognition
-PyAudio
-wikipedia
-smtplib
